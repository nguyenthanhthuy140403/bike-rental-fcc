# Bike Rental Shop
A Bash and PostgreSQL bike rental application built as part of the freeCodeCamp Relational Database certification.

## Features
- Rent a bike
- Return a bike
- Store customer information
- Track bike availability
- Manage rentals using PostgreSQL

## Technologies
- Bash
- PostgreSQl
- Git
- GitHub
